El Pollo Loco

"El Pollo Loco" is an action-packed game where players control a character using arrow keys to navigate forward and backward.
The game features dynamic gameplay with the following mechanics:

Movement: Navigate through the game using the arrow keys. Bottle Throw (Key D): Pressing the 'D' key allows players to throw bottles with a 1-second cooldown. Use this skill strategically. Jump (Space): The 'Space' key enables the character to jump, essential for avoiding obstacles and engaging enemies.

Enemies: Encounter two types of adversaries – smaller chickens and a mosquito army. Defeat them by jumping on them.

Boss Battle: Face the final challenge – a colossal chicken. Initially, attack it with bottles, but beware, taking damage increases its speed. Stay mobile and maintain a safe distance. Enraged Mode: As the boss nears defeat, it enters an enraged state. Although smaller, it becomes faster and deals triple damage. Avoid getting hit; it's more effective to defeat it by jumping on top. Be prepared for multiple attempts as it won't die instantly from a single jump.
